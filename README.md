# Vue File Manager 

[![Vue](https://img.shields.io/badge/Vue-3.5-green)](https://vuejs.org/)
[![Vite](https://img.shields.io/badge/Vite-5.4-blue)](https://vitejs.dev/)
[![Tailwind CSS](https://img.shields.io/badge/Tailwind-3.4-blue)](https://tailwindcss.com/)

Современный файловый менеджер с интуитивно понятным интерфейсом, созданный на Vue 3. Поддерживает drag-and-drop загрузку, переключение между видами, фильтрацию, сортировку и статистику файлов.



##  Основные возможности

*   **Два режима просмотра**: Карточки и таблица
*   **Drag & Drop загрузка**: Перетаскивайте файлы прямо в окно браузера
*   **Поиск и фильтрация**: Поиск по имени и фильтрация по типу файлов
*   **Сортировка**: По имени, размеру и дате создания 
*   **Пакетные операции**: Выбор нескольких файлов и массовое удаление
*   **Поддержка тем**: Переключение между светлой и тёмной темой
*   **Статистика**: Полезная сводка по файлам 
*   **Предпросмотр**: Быстрый просмотр выбранного файла
*   **Адаптивный дизайн**: Корректно отображается на мобильных устройствах

##  Быстрый старт

### Установка зависимостей
```
bash
npm install
```
Запуск в режиме разработки
```
bash
npm run dev
```
Приложение будет доступно по адресу http://localhost:5173.

## Использование

Добавление файлов:

Нажмите кнопку "Добавить"

Или перетащите файлы в специальную зону "Drag & Drop загрузка"



### Управление файлами:

Поиск: Введите текст в верхнем поле поиска

Фильтрация: Выберите тип файла в выпадающем списке справа

Сортировка: Используйте выпадающие списки в правом верхнем углу

Просмотр: Кликните на иконку файла или его имя для предпросмотра

Удаление: Нажмите кнопку "Удалить" на карточке файла или в таблице

Скачивание: Нажмите кнопку "Скачать" 



### Массовые операции:

Отметьте чекбоксы рядом с файлами для выбора

Используйте появившуюся кнопку "Удалить выбранные"




## Основная логика содержится в FileManager.vue:

Реактивное состояние: Управляется через Composition API

Сортировка и фильтрация: Реализованы через вычисляемые свойства

Drag & Drop: Нативная реализация с помощью событий браузера

Статистика: Автоматически пересчитывается при изменении списка файлов




## Tailwind CSS
### Конфигурация в tailwind.config.js включает обработку файлов .vue и .js:

### javascript
```
export default {
  content: ['./index.html','./src/**/*.{vue,js}'],
  theme: { extend: {} },
  plugins: []
}
```
### Vite
Простая конфигурация с поддержкой Vue:
```
javascript
import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'
export default defineConfig({ plugins: [vue()] })
```
